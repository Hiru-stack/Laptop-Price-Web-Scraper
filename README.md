# Laptop Price Web Scraper

A Python-based web scraping project developed using BeautifulSoup, Requests, and Pandas to collect laptop information from an e-commerce website.

## 📌 Project Overview
This project automatically extracts laptop data such as:
- Laptop Title
- Price
- Description
- Ratings

The scraped data is stored in:
- CSV format
- JSON format

This project was completed as part of a Data Science Internship Task.

---

## 🛠️ Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas
